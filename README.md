# LeCinema

TMDb API destekli, modern film ve dizi keşif platformu.

## Özellikler
- Film ve dizi detayları (oyuncu kadrosu, fragmanlar, benzer içerikler).
- Kategori bazlı listeleme (Aksiyon, Animasyon, Popüler vb.).
- Responsive tasarım.

## Kurulum
```bash
git clone https://github.com/furkanusakpinar/Cinema-app.git
cd Cinema-app
npm install
npm run dev
```

---
*TMDb API kullanılarak geliştirilmiştir.*
